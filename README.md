# simple-react-calendar

## Install

```bash
yard add simple-react-calendar
```

## Usage

```jsx
import CalendarForm from 'simple-react-calendar'

const SomeFunctionalComponent = () => {
    return (
        <div>
            <CalendarForm />
        </div>
    )
}
```
